# PLENGER-RACING

## Description
PLENGER-RACING is a thrilling 3D racing game developed in Python. Players can race against each other with uniquely skilled cars in various exciting environments.

## Installation Instructions
1. Ensure you have Python installed on your machine. You can download it from [python.org](https://www.python.org/downloads/).
2. Clone the repository:
   ```bash
   git clone https://github.com/khaidananggara-lang/PLENGER-RACING.git
   ```
3. Navigate into the project directory:
   ```bash
   cd PLENGER-RACING
   ```
4. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Game Controls
- **Movement:** WASD or Arrow keys
- **Nitro:** SPACE
- **Brake:** S
- **Exit:** ESC

## Gameplay Features
- Auto-driving feature for ease of use.
- Choose from 3 cars, each with different skills (Speed King, Drift Master, Tank).
- Nitro boost system for speed bursts during races.

## Car Descriptions
- **Speed King:** A car designed for speed, perfect for straight tracks.
- **Drift Master:** Ideal for tight corners and high-agility maneuvers.
- **Tank:** Heavy built for durability, but not as fast as others.

## 3D Model Resources
The 3D models used in the game are sourced from:
- [Sketchfab](https://www.sketchfab.com)
- [TurboSquid](https://www.turbosquid.com)

## How to Run the Game
To start the game, navigate to the project folder and run:
```bash
python main.py
```
